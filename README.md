# CRUD_Node.js_Mysql
## Belajar CRUD(create, read, update, delete) dengan node.js dan mysql

### Menjalankan program:
- Pastikan node.js & npm nya sudah terinstall
- jalankan xampp
- buka terminal
- ketik : node [nama file ektensi .js]
- contoh : node read.js, lalu enter
